# Skin-care-analysis
Frontend web app that recommends skincare based on skin type and concerns. Users select inputs, and JavaScript maps them to ingredients with explanations. Built using HTML, CSS, and JS with a clean, responsive UI and basic login/signup flow.

## 📌 Overview

This is a frontend-based web application that provides personalized skincare recommendations based on a user's skin type and specific skin concerns. The application helps users understand which ingredients are suitable for their skin and why they should use them.

---

## 🚀 Features

* Select skin type (Dry, Oily, Normal, Combination, Sensitive)
* Choose multiple skin concerns (Acne, Pigmentation, Aging, Dullness, etc.)
* Get ingredient-based recommendations
* Explanation for each recommended ingredient
* Clean and responsive user interface
* Simple and user-friendly design

---

## 🛠️ Tech Stack

* **HTML** – Structure of the web pages
* **CSS** – Styling and layout
* **JavaScript** – Logic and dynamic content handling

---

## 🧠 How It Works

1. User selects their skin type.
2. User selects one or more skin concerns.
3. The application processes the input using predefined mappings.
4. It displays suitable skincare ingredients.
5. Each ingredient comes with a short explanation of its benefits.
